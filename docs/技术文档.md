## 1. 项目概述

### 1.1 项目简介
诗词大全是一个基于 Spring Boot 3.x 构建的古典诗词管理与AI赏析平台，提供诗词检索、分类浏览、AI智能赏析等核心功能。

### 1.2 技术特点
- 采用前后端分离架构（前端纯静态HTML，后端RESTful API）
- 支持多种主流AI大模型（OpenAI、文心一言、通义千问、讯飞星火）
- 异步AI分析，避免阻塞用户操作
- 全文检索能力，支持诗词内容搜索

#### 4.2.2 工厂模式统一管理

```java
@Component
public class AiServiceFactory {
    @Autowired
    private List<AiServiceProvider> providers;
    
    private final Map<String, AiServiceProvider> providerMap = new ConcurrentHashMap<>();
    
    @PostConstruct
    public void init() {
        for (AiServiceProvider provider : providers) {
            providerMap.put(provider.getProviderName(), provider);
        }
    }
    
    public AiServiceProvider getProvider(String providerName) {
        return providerMap.get(providerName);
    }
    
    public AiServiceProvider getDefaultProvider() {
        // 从配置获取默认模型
        return getProvider(aiConfig.getDefaultModel());
    }
}
```

#### 4.2.3 异步处理机制

```java
@Service
public class PoetryAnalysisServiceImpl implements PoetryAnalysisService {
    
    @Async
    protected void doAsyncAnalysis(PoetryAnalysis analysis, Poetry poetry, String modelName) {
        try {
            // 调用AI服务
            String result = provider.analyzePoetry(poetry);
            
            // 解析结果
            parseAndSaveResult(analysis, result, provider);
            
            // 更新状态为完成
            analysis.setStatus(1);
        } catch (Exception e) {
            // 更新状态为失败
            analysis.setStatus(2);
            analysis.setErrorMsg(e.getMessage());
        }
        updateById(analysis);
    }
}
```

## 5. 切面编程(AOP)

### 5.1 审计日志

**自定义注解**:
```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface AuditLog {
    String operation();  // 操作描述
}
```

**切面实现**:
```java
@Aspect
@Component
public class LogAspect {
    @Around("@annotation(auditLog)")
    public Object around(ProceedingJoinPoint point, AuditLog auditLog) {
        // 记录操作日志
        log.info("[审计日志] 操作: {}, 方法: {}", 
            auditLog.operation(), 
            point.getSignature().getName());
        return point.proceed();
    }
}
```

### 5.2 接口耗时统计

**自定义注解**:
```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface TimeCost {
    String value();           // 方法描述
    TimeUnit unit();          // 时间单位
}
```

**切面实现**:
```java
@Aspect
@Component
public class TimeCostAspect {
    @Around("@annotation(timeCost)")
    public Object around(ProceedingJoinPoint point, TimeCost timeCost) {
        long start = System.currentTimeMillis();
        Object result = point.proceed();
        long cost = System.currentTimeMillis() - start;
        log.info("【耗时统计】方法: {}, 耗时: {}ms", 
            timeCost.value(), cost);
        return result;
    }
}
```
